# NyaaAndroid

A modern, native Android client for **Nyaa.si**, built with **Kotlin** and **Jetpack Compose**.
This application allows users to browse, search, and view details of torrents directly from a mobile-friendly interface.

- **Browsing & Search**: Search for torrents with advanced category filtering (Anime, Audio, Literature, Live Action, etc.).
- **Pagination**: Seamless navigation through multiple pages of results.
- **Detailed View**: View torrent descriptions (HTML rendered), file size, submitter, seeds/leechers, and comments.
- **Magnet Integration**: One-tap access to open magnet links with your favorite torrent client.
- **Visual Indicators**:
  - Distinct icons for different categories.
  - Specific Kanji display for "Raw" categories.
  - Color-coded badges for easy identification.
- **Theming**:
  - Light Mode
  - Dark Mode (Grey)
  - **AMOLED Mode** (True Black for battery saving on OLED screens)
  - System Default
